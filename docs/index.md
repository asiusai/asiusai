# Asius AI

## Backends

### Comma 

[Comma API](https://api.comma.ai), can be used with [comma.asius.ai](https://comma.asius.ai), default with any comma device

### Konik 

[Konik API](https://api.konik.ai), can be used with [konik.asius.ai](https://konik.asius.ai), default with any konik device

### Asius

[Asius API](https://api.asius.ai), can be used with [connect.asius.ai](https://connect.asius.ai), you have to install one of our forks (openpilot or sunnypilot)



## Openpilot fork

## Sunnypilot fork

## Benefits for you

Depends on how much you use us

### just using as a new frontend, using comma(or konik) apis, and other forks (use comma.asius.ai or konik.asius.ai)
- better file management
- nicer ui
- playback ui
- web app
- android app

### Using our forks
- Video live streaming
- Joystick from the app
- Edit device params from the app
- Change models from the app
- Edit navigation favourites and navigate (sunnypilot only)
- connect to device with bluetooth support (soon)


### Using our APIs (10 usd a month, with 2 week trial)
- High quality video streaming
- Video clip rendering
- 1TB of upload capacity per device
- route OG images
- no blocking devices, use your device however you like, we don't block on devices
- use comma clones or comma zero or any device you want to
- end to end encryption (maybe soon)
